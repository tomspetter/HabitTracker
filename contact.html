Frequently Asked Questions

Is ChainOfDots really free?

Yes! ChainOfDots is completely free and open source under the MIT license. You can use it, modify it, and even self-host it without any cost.

Can I self-host ChainOfDots?

Absolutely! Check the README for installation and deployment instructions.

Is my data secure?

Yes. ChainOfDots uses bcrypt password hashing, CSRF protection, rate limiting, and secure sessions. You can also export your data at any time.

Can I contribute to the project?

Yes! Contributions are welcome. Check the GitHub repository to get started.

How do I report a bug?

Open an issue on GitHub Issues with a description of the bug and steps to reproduce it.

Will there be a mobile app?

ChainOfDots is mobile-responsive and works great in mobile browsers. A native mobile app is not currently planned, but the project is open source‚Äîfeel free to build one!

How can I support the project?

You can buy me a coffee, star the project on GitHub, share it with others, or contribute code!

Need Direct Contact?

For general inquiries, please use GitHub Issues so the community can benefit from the discussion.

For private matters, you can reach out via GitHub profile contact options.
